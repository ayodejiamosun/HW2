MY VIRTUAL LIBRARY

Check out My Virtual Library!

Challenge: Create a web application that utilizes a database to retriev and store data.

Insight: As a reader, I want to have my own virtual library, so that I can keep track of the books that I have read and plan to read. This would be useful considering the fact that I may not own a physcial copy of every book I've read.

Idea: A web application that allows the user to create an account and search for books via GoodReads. The user would then be able to choose to add the book to either their Queue of "To Read" Books or a Library of "Read" books. The user would also be able to update books in Queue to their Library once they have completed the book.

Technologies Used: Node.js, Express.js, MySQL, Sequelize
