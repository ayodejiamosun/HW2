EMPLOYEE MANAGEMENT SYSTEM

Check out Employee Management System!

Challenge: Build a command-line application that allows the user to enter and retrieve information, regarding employees, from a database.

Insight: As a business owner, I want to be able to view and manage the departments, roles, and employees in my company, so that I can organize and plan my business

Idea: A command-line application, with an easy interface that non-developers can view and interact with, that stores employee information in databases. User will be allowed to add, view, delete, and update departments, roles, and employees information.

Technologies Used: Node.js, Inquirer, MySQL
